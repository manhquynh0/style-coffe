## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/coffee-style.git
```

Install dependencies:

```bash
npm install
```
Start Tailwind CSS: yarn build:css

Run the project using **Live Server** in VS Code.
